General Cargo

PF_CC_GENERAL_CARGO

Cargo description Most cargos can be transported packaged in crates, boxes, barrels, jars, tins, cans, bags, sacks, bales, or pallets. This class is suitable for nearly all cargos, including bulk, liquid, gas, powder, and food items.
Example cargos Aluminium, building materials, chemicals, china clay, coal, copper, farm supplies, fish, food, fruit, glass, goods, grain, iron ore, logs, oil, oxygen, paper, steel, timber, tyres, wool, zinc and more.
Cargo GRF advice
  • This is the most widely supported cargo class in vehicles and should be set for almost every freight cargo to maximise vehicle compatibility.
  • For truly unique cargos, such as radioactive materials, which require dedicated vehicles, it might make sense to not set this class.
  • Don't set for non-freight cargos like passengers and mail.
Vehicles description General cargo vehicles, including flat vehicles, vehicles with sides, and vehicles with roofs.
Example rail vehicles Box car, container wagon, goods van, open wagon
Example road vehicles Box truck, container truck, curtain side truck, general cargo truck
Example ships Container ship, general cargo vessel, pallet carrier
Vehicle GRF advice
  • Freight vehicle GRFs: always provide vehicles supporting this class as it represents the most common class for freight cargos.
  • Does not apply if there are no freight vehicles in the GRF.
Replaces or reuses NML/GS classname CC_PIECE_GOODS
Bit number in legacy scheme 5

Dry Powder

PF_CC_POWDER

Cargo description Dry powdered materials that flow under air pressure.
Example cargos Carbon black, cement, cleaning agents, flour, quicklime, sand, soda ash, sugar and more.
Cargo GRF advice

Set this class for all powder cargos, including food-grade and non-food-grade types. Powders are relatively niche, and food-grade powders are a niche within a niche, not justifying a separate class.
Vehicles description Vehicles designed for transporting powdered materials.
Example rail vehicles Bulk powder trucks, cabbage, pneumatic tankers, silo wagons
Example road vehicles Cabbage
Example ships Cabbage
Vehicle GRF advice

If specialist powder vehicles need to be separated from general powder vehicles, manage this by including or excluding specific cargo labels. For example, GRFs may provide dedicated vehicles for food-grade powders or carbon black, which should be handled with appropriate labels.
Replaces or reuses NML/GS classname CC_POWDERIZED
Bit number in legacy scheme 11
